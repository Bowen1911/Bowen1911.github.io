这个博客记录 Docker 使用笔记。因为 Docker 有很多使用细节，这里只记录用于科研及基本工作的使用经验。

详细使用手册：https://docker-practice.github.io/zh-cn/basic_concept/

# 基本概念

Docker 可以方便复现项目，一个 Docker 仓库通常包括一个镜像和若干容器，镜像就像面向对象编程的类，容器就像类的方法。

# 网络问题

国内无法直接连接官方 Docker 仓库。

使用代理 `registry.yhnotes.com/`：

官方镜像

```bash
# 原始命令
docker pull nginx:latest

# 使用代理
docker pull registry.yhnotes.com/library/nginx:latest
```

非官方镜像

```bash
# 原始命令
docker pull bitnami/redis:latest

# 使用代理
docker pull registry.yhnotes.com/bitnami/redis:latest
```

参考 https://www.yhnotes.com/blog/dockerhub-proxy/
